# 基于SSM酒店预订系统

#### 介绍
管理员角色包含以下功能：管理员登录,用户管理,楼层管理,房型管理,房间管理,客户管理,预定管理,入住管理,营业额统计等功能。
用户角色包含以下功能：查看客房,用户登录,预定酒店,查看我的订单,个人资料修改等功能。
技术点： HTML+CSS+JavaScript+jsp+mysql+Spring+SpringMVC+mybatis
